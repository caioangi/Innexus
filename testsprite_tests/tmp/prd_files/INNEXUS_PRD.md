# PRD — INNEXUS Landing Page

## Visão Geral
- Projeto: Landing page premium em dark mode unificado
- Objetivo: posicionar marca e converter via “Agendar Reunião”
- Estilo: Linear, Vercel, Raycast, 21st.dev
- Stack: React + TypeScript + Vite + Tailwind

## Escopo
- Hero, Posicionamento, O que Criamos (grid 3×4), Diferenciais (3 cards), Processo (5 passos), Projetos, Planos, CTA Final, Footer.

## Requisitos de UX
- Fundo contínuo `#0b0b0b` sem blocos claros
- Títulos `#f2f2f2`, subtítulos `#bdbdbd`, textos `#8c8c8c`
- Bordas finas `#1a1a1a`, hover linear sutil, animações suaves

## Não Escopo
- Backend real, CRM, CMS, área logada

## Métricas
- Cliques em “Agendar Reunião”, scroll até Planos, interações nos Projetos

## Critérios de Aceite
- Sem overlay de erro, tipografia HERO 48–56px desktop, grids e CTAs funcionais, Lighthouse A11y/SEO adequados
